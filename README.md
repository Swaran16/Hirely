# Job AI Platform

A comprehensive platform featuring a Next.js frontend, Express.js backend, and a Chrome extension for job assistance.

## Structure

- `/frontend` - Next.js App Router (pages)
- `/backend` - Express.js API
- `/extension` - Chrome Extension (placeholder)

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. Configure `.env` based on `backend/.env` template
4. `npm run dev` (requires setting up a dev script)

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
